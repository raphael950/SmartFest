# Circuits GPS

Ce dossier contient les fichiers GPX des circuits utilisés dans SmartFest.

## Structure

- `circuit_du_mans.gpx` - Fichier GPX du circuit Bugatti - Le Mans

## Utilisation

Les fichiers GPX sont parsés par le service `app/services/gpx_parser.ts` qui les convertit en coordonnées SVG pour l'affichage sur la page live-timing.
